## Описание проекта

Этот сайт создан с помощью статического генератора Hugo. Особенности:

- Использована тема Bearblog
- Автоматическое развертывание через GitHub Pages
- Поддержка Markdown

```bash
# Команда запуска локального сервера
hugo server -D
```
